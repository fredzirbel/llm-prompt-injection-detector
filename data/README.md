# Dataset Sources

## Primary: deepset/prompt-injections (HuggingFace)
- URL: https://huggingface.co/datasets/deepset/prompt-injections
- License: MIT
- Contains labeled prompt injection and benign samples
- Downloaded via `python data/download_data.py`

## Supplemental: Curated samples
- `injection_samples.csv` — manually curated injection patterns
- `benign_samples.csv` — normal user prompts
- Patterns informed by OWASP LLM Top 10 and published research

## Combined dataset
- `combined_dataset.csv` — generated by `download_data.py`, merges all sources
- Used by `ml/train.py` for classifier training

## References
- OWASP Top 10 for LLM Applications: https://owasp.org/www-project-top-10-for-large-language-model-applications/
- Perez & Ribeiro, "Ignore This Title and HackAPrompt" (2023)
- Greshake et al., "Not what you've signed up for" (2023)
